# BankIQ

AI Powered Banking Intelligence Platform

## Features

- JWT Authentication
- Dashboard Analytics
- Add Transaction
- Edit Transaction
- Delete Transaction
- Search Transactions
- CSV Upload
- CSV Export
- Insights
- Pie Charts
- Monthly Analysis

## Tech Stack

Frontend

- React
- Axios
- Recharts

Backend

- FastAPI
- SQLAlchemy
- JWT

Database

- PostgreSQL

Libraries

- Pandas
- Recharts

## Backend

```bash
uvicorn app.main:app --reload
```

## Frontend

```bash
npm run dev
```
